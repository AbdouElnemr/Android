\subsection{Detailed Description}
\hyperlink{class_communication_lib}{Communication\-Lib} class. 

This class allows to comunicate Unity with an external call thorught datagrams packet U\-D\-P. The message to send must to be of the string type and the Ip is ever localhost (\char`\"{}127.\-0.\-0.\-1\char`\"{}).
